Cheat Sheet

Common CSS properties

Alek Howland

Property Name Description
font-size Font Size Used to set text sizes
background-color Background Color Controls the background color of elements.
opacity Opacity Used to control the transparency of an element.
The value of this property ranges from 0 (transparent) to 1 (opaque).
font-weight Font Weight Used to set the weight (boldness) of text.
The provided value can be a keyword such as bold or normal.
text-align Text Align Used to set the text alignment of inline contents.
This property can be set to these values: left, right, or center.
border-collapse Border-collapse Sets whether cells inside a <table> have shared or separate borders.
padding Padding Sets the padding area on all four sides of an element at once.
margin Margin Sets the margin area on all four sides of an element.
